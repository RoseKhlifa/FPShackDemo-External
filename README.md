## English

**FPSHackDemo-External**  
A basic external cheat framework designed for FPS games,  
including ESP rendering and Aimbot functionality.  
Great starter project for learning game hacking and external memory tools.

| Module | Description |

| Aimbot | Closest to Crosshair, Distance-based, FOV-based |
| Warning System | Direction triangle & distance alert |
| Radar | Multiple radar rendering modes |
| Line ESP | Four RGB line styles pointing targets |
| Info Overlay | HP bar, name, distance and more |
